# VGEditor

An editor component for the graph.

## Usage

```html
<template>
  <v-g-editor>
    <flow/>
  </v-g-editor>
</template>

<script>
import VGEditor, { Flow } from 'vg-editor'
export default {
  components: { VGEditor, Flow }
}
</script>
```
